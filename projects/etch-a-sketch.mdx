 *Etch-A-Sketch*

*Javascript
* HTML
* CSS

A fun project I did using vanilla Javascript. The user can drag their mouse over the pixels to 
create an Etch-A-Sketch like drawing. The color picker changes the pixel colors. The user can
also change the size of the pixels and clear the board.

https://shannonnozomi.github.io/etch-a-sketch/
